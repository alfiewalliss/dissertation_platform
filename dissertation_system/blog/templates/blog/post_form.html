Tags

What are tags?

We understand you may have some questions about tags and how they work on our platform. Tags are labels that help categorize users and posts so that people can search for and discover content relevant to their interests. For example, if someone tags their post as 'Computer Science', then a user looking for computer science related content can look for posts with that tag.
Posts must have one primary tag, but they can have multiple secondary tags. On the other hand, users can have zero or many tags. If a user has a tag that matches a post's tag, then the user is able to peer review it if they are verified. This is a great way to ensure quality and accuracy in the content posted. When making a post, you should think carefully about the tags you will add as this will effect the users able to review it.
